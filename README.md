# Blog-Post-Management-Website
Personal Student Project


Technical Requirements
- Backend: Node.js + Express.js
- Database: MongoDB + Mongoose
- Templating: Pug
- Styling: Bootstrap 5
- Validation: Joi

Overview
In this practical assignment, you will build a blog post management system where users can register, 
log in, create, edit, delete, and view blog posts, and search posts by title or tags. Only the user who 
created a post can modify it. You must use Pug for rendering views and Bootstrap 5 for styling.
